📦 Requirements & Installation

To run the generated code, install these packages:

# Python
pip install pandas numpy scikit-learn matplotlib seaborn

# For Jupyter notebooks
pip install jupyter notebook ipython

# R (if using R export)
install.packages(c("tidyverse", "caret", "randomForest", "ggplot2"))

Quick Start:

1. Save the code to a .py (Python) or .ipynb (Jupyter) or .R file

2. Replace 'your_data.csv' with your actual file path

3. Run: python script.py
